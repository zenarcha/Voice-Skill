# Anti-Patterns

An append-only library of **real, annotated examples** of writing that reads as cringe or AI-slop *to you*. The short, always-on *rules* live in [guardrails.md](guardrails.md); this file is the growing evidence behind them — examples to calibrate against and to consult when updating the skill. It is read for calibration, not on every draft.

**Why this file exists:** inference alone isn't enough for "what makes me cringe." The patterns are easiest to name when there's a concrete specimen in front of you. Add to this file whenever you spot something that makes you recoil.

**How the skill uses it:** as a negative reference — examples of what *not* to produce. It never copies anything from here; it uses each entry's *why it fails* note to avoid the same move.

The onboarding interview seeds the first one or two entries from your "what reads as fake" answers. Everything after that, you add over time.

---

## Entry format

Copy this block for each new example.

```
### [short label] — [date]
**Source / where seen:** [link or context]

**The example:**
> [paste the offending hook / line / post]

**Why it fails (for me):**
- [the specific move that makes it cringe — tone, cliché, posture, structure]

**Maps to rule:** [which guardrails.md rule this is evidence for, if any]
```

## Examples

> One generic specimen is shown below to illustrate the format. Replace or add your own — the entries that matter are the ones in *your* taste, captured during onboarding and as you go.

### Generic slop hook (illustrative) — template
**Source / where seen:** illustrative example, not a real find

**The example:**
> 🚀 This one framework is the secret weapon every professional needs. Here's the system that changed how I think about success. Let's dive in. 👇

**Why it fails (for the example writer):**
- Emoji in the opening line.
- "secret weapon" — power-word cliché.
- "every professional needs" — sweeping universal claim.
- "the system that changed how I think" — manufactured authority; certain tone.

**Maps to rule:** guardrails.md §A (emojis), §F (clichés, sweeping claims, authoritative tone).
